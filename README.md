# Experiments

Collection of browser-based experiments in psycholinguistics, pragmatics, and sociology.

🔗 **Live site:** https://kgmbh.github.io/experiments/

---

## Overview

This repository contains a simple GitHub Pages site that serves as an entry point to interactive online experiments.

The experiments themselves are hosted externally (Netlify) and can be accessed via the links on the site.

---

## Categories

### Psycholinguistic / Pragmatics
- Experiments on expressive meaning, interpretation, and language processing

### Sociological
- Questionnaires and studies on attitudes, gender roles, and social evaluation

---

## Tech

- HTML / CSS (GitHub Pages)
- Magpie (experiment framework)
- Netlify (experiment hosting)
